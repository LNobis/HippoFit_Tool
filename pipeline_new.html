Image pre-processing pipeline


MR images from UK Biobank have been preprocessed with an automated pipeline using FMRIB Software Library (FSL), published previously by the FMRIB Analysis Group here.


The analysis is described in more detail in:

Alfaro-Almagro, F., Jenkinson, M., Bangerter, N. K., Andersson, J. L. R., Griffanti, L., Douaud, G. E., … Smith, S. M. (2018). Image processing and Quality Control for the first 10,000 brain imaging datasets from UK Biobank. NeuroImage, 166, 400–424. DOI: 10.1016/j.neuroimage.2017.10.034



Volume analysis pipeline

After obtaining numeric values for the volume of the hippocampus as described in Alfaro-Almagro et al (2018), we prepared the data for use as normative values as described here. This analysis includes correction for outliers and confounds. Please refer to the background section of this website, and to our preprint, for detailed methods.
